- # 个人作品集网站 (Portfolio) - 产品需求文档 (PRD)

  ## 1. 文档概述与产品定位
  - **产品名称**：个人作品集展示网站 (Personal Portfolio)
  - **产品目标**：打造一个具备极简美学、现代感极强的纯静态个人主页，用于向潜在客户或雇主展示个人专业能力、过往项目经验及技术栈。
  - **设计基调**：深色模式 (Dark Mode)、左右分栏布局、玻璃拟物化/悬浮卡片质感、平滑流畅的交互体验。

  ## 2. 整体页面布局 (Desktop 端)
  网站采用 **单屏左右分栏布局 (Split-Screen Layout)**，避免传统的长滚动页面，提供类似画廊的浏览体验。
  - **顶部导航 (Navbar)**：
    - 左侧：网站 Logo 或纯文本 "PORTFOLIO"。
    - 右侧：锚点导航链接（Work, About, Contact）。
  - **左侧栏 (Left Panel - 固定区域)**：占据屏幕左侧约 35%-40% 宽度，滚动右侧内容时左侧保持固定 (Sticky/Fixed)。
  - **右侧栏 (Right Panel - 滑动区域)**：占据屏幕右侧约 60%-65% 宽度，作为核心项目的动态展示区。

  ## 3. 核心功能与模块详情

  ### 3.1 左侧个人信息区 (Fixed Info Panel)
  用于展示个人的核心身份标识与关键数据，传达专业形象。
  - **个人名片**：
    - 头像：圆形或圆角矩形的高清个人头像。
    - 基本信息：姓名、年龄、所在地 (Based in)、职位角色 (Role: 比如 Creative Web Developer)。
  - **主视觉文案 (Hero Section)**：
    - 大标题 (H1)：例如 "Crafting Digital Experiences"。
    - 简短自述：一段关于个人开发理念的精炼文字（例如：专注于创建优雅且实用的数字产品...）。
  - **数据统计 (Stats)**：
    - 以显眼的大数字加小标题形式展示核心指标，例如：`[03] PROJECTS`、`[12+] YEARS`、`[08+] SKILLS`。

  ### 3.2 右侧项目展示区 (Interactive Showcase)
  用于详细展示 3 个核心项目，结合背景图与卡片轮播，营造强烈的视觉冲击力。
  - **背景层**：
    - 支持全屏铺满的高清背景图片（从准备好的 3 张素材中选择，或根据滑动的项目动态切换背景）。
    - 背景需增加深色遮罩 (Dark Overlay) 或渐变层，确保前景文字和卡片清晰可见。
    - 顶部标题："FEATURED PROJECTS"。
  - **项目卡片轮播 (Swiper Carousel)**：
    - **交互方式**：支持鼠标拖拽横向滑动、键盘左右键切换、底部导航点/箭头点击切换。一屏内完整展示 1 个项目卡片，并露出下一个卡片的边缘，暗示可滑动。
    - **卡片设计 (Project Card)**：采用半透明或纯色圆角卡片设计（带轻微阴影隔离背景）。
    - **卡片内容结构**：
      1. **项目截图**：展示项目界面（从准备好的 3 张项目图中提取）。
      2. **项目标题**：清晰的项目名称（如：AI ToDo List - 鱼皮的待办清单）。
      3. **项目简介**：1-2 句话的功能描述。
      4. **技术栈标签 (Tech Stack)**：展示使用的核心技术（如 Vue.js 3 | Pinia | Node.js）。
      5. **状态/分类标签**：展示项目进度或特点（如 "Done", "In Progress" 等小 Tag）。
      6. **发布时间**：如 2026-01-12。

  ### 3.3 响应式设计 (移动端适配 Mobile Responsiveness)
  考虑到手机屏幕尺寸，在屏幕宽度小于 `768px` (Tablet/Mobile) 时，需改变布局策略：
  - **布局转换**：从左右分栏转换为**上下垂直堆叠**。
  - **上方**：展示原左侧的个人信息区（头像、标题、统计数据），取消固定 (Fixed) 状态，变为可自然滚动。
  - **下方**：展示原右侧的项目轮播区，卡片宽度占满屏幕，保留左右滑动功能。

  ## 4. 技术与交互要求
  - **前端技术**：纯静态 HTML/CSS/JS 或使用轻量级框架（如 Vue/React 搭配 Vite），推荐使用 **Tailwind CSS** 以快速实现复杂的布局和深色 UI。
  - **滑动组件**：强烈建议引入成熟的轮播库（如 **Swiper.js**）来处理右侧卡片的物理滑动阻尼、分页器以及边缘露出效果。
  - **动画效果**：
    - 页面初次加载时，左侧文字和数据统计带有淡入向上 (Fade In Up) 的动效。
    - 鼠标悬停 (Hover) 在项目卡片时，卡片有轻微的上浮和阴影加深效果。
